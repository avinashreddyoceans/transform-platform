- `ON_OPEN` — fires when the window opens
- `ON_CLOSING` — fires when the window closes
- `RECURRING_WHILE_OPEN` — fires on `recurringInterval` while window is open
- `ON_EMPTY_CLOSE` — fires at close if zero events were collected
- `ON_THRESHOLD_REACHED` — fires when event count crosses a configured threshold
- `ON_ERROR` — fires when any action in the chain fails
- `ON_MANUAL_TRIGGER` — fires only via explicit API call
- `ON_DUPLICATE_DETECTED` — fires when a duplicate event is rejected

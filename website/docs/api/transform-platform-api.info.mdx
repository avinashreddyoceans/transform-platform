Enterprise-grade, spec-driven file ↔ event transformation engine.

Transform Platform exposes a REST API for managing **FileSpecs**, triggering
**file-to-events** and **events-to-file** pipelines, managing **client integrations**
(SFTP, Kafka, REST, S3), and monitoring platform health.

## Authentication
All endpoints require a Bearer token passed in the `Authorization` header:
```
Authorization: Bearer <your-token>
```

## Error Format
All error responses follow the RFC 7807 Problem Details format:
```json
{
  "type": "https://transform-platform/errors/spec-not-found",
  "title": "FileSpec Not Found",
  "status": 404,
  "detail": "No FileSpec found with id 'acme-payments-v2'",
  "instance": "/api/specs/acme-payments-v2"
}
```
